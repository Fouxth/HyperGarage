# การวิเคราะห์ระบบ (System Analysis)

## ภาพรวมปัญหา

การเลือกซื้ออะไหล่แต่งรถยนต์ในปัจจุบันมีความยุ่งยากสำหรับผู้ซื้อ เนื่องจากอะไหล่แต่ละชิ้นมีความเฉพาะเจาะจงต่อรุ่นเครื่องยนต์ (เช่น ก้านสูบสำหรับ B16A ใช้กับ B18C ไม่ได้) ร้านค้าออนไลน์ทั่วไปไม่มีระบบตรวจสอบความเข้ากันได้อัตโนมัติ ทำให้ลูกค้าสั่งผิดรุ่นบ่อย

ระบบ HyperGarage มุ่งแก้ปัญหานี้โดยพัฒนาระบบ Compatibility Checker ที่กรองสินค้าตามข้อมูลรถยนต์ของลูกค้าได้ทันที และมีระบบหลังบ้านที่จัดการสต็อก/คำสั่งซื้อได้อย่างเป็นระบบ

---

## ความต้องการด้านฟังก์ชันการทำงานหลัก (Functional Requirements)

### กลุ่มที่ 1: ลูกค้า (Customer Frontend)

| รหัส | ฟังก์ชัน | รายละเอียด |
|------|----------|------------|
| CF-01 | Compatibility Checker | เลือกยี่ห้อรถ > รุ่น > เจเนอเรชัน > เครื่องยนต์ เพื่อกรองเฉพาะอะไหล่ที่เข้ากันได้กับรถของลูกค้า ข้อมูลดึงจากฐานข้อมูลรถยนต์จริง |
| CF-02 | Product Browsing | ค้นหา กรอง และเรียงลำดับสินค้าตามราคา/คะแนน/หมวดหมู่/แบรนด์ พร้อมระบบ pagination |
| CF-03 | Guest Checkout | สั่งซื้อสินค้าโดยไม่ต้องสมัครสมาชิก ใช้เบอร์โทรศัพท์เป็นตัวระบุตัวตน ระบบตัดสต็อกจริงผ่าน transaction |
| CF-04 | Order Tracking | ติดตามสถานะคำสั่งซื้อด้วยเบอร์โทรศัพท์ (pending → processing → shipped → delivered) |
| CF-05 | Product Reviews | เขียนรีวิวสินค้าพร้อมให้คะแนน (1-5 ดาว) คะแนนเฉลี่ยและจำนวนรีวิวคำนวณแบบ real-time |
| CF-06 | Shopping Cart | เพิ่ม/ลบสินค้าในตะกร้า เก็บข้อมูลใน localStorage ไม่หายแม้ปิดเบราว์เซอร์ |
| CF-07 | Wishlist | บันทึกสินค้าที่สนใจไว้ดูภายหลัง |
| CF-08 | Flash Sale & Promotions | ดูสินค้าลดราคาพิเศษ + คูปองส่วนลดที่ใช้ได้ |

### กลุ่มที่ 2: ผู้ดูแลระบบ (Admin Back-office)

| รหัส | ฟังก์ชัน | รายละเอียด |
|------|----------|------------|
| AB-01 | Product CRUD | เพิ่ม แก้ไข ลบสินค้า พร้อมจัดการราคา/สต็อก/รูปภาพ/สเปค ป้องกันลบสินค้าที่มีคำสั่งซื้อแล้ว (Restrict) |
| AB-02 | Category & Brand Management | จัดการหมวดหมู่และแบรนด์สินค้า ป้องกันลบหมวดหมู่/แบรนด์ที่มีสินค้าอยู่ |
| AB-03 | Inventory Management | ตรวจสอบสต็อกสินค้า แจ้งเตือนสินค้าใกล้หมด อัปเดตจำนวนสต็อก |
| AB-04 | Vehicle Taxonomy | จัดการข้อมูลรถยนต์แบบ hierarchy: ยี่ห้อ > รุ่น > เจเนอเรชัน > เครื่องยนต์ |
| AB-05 | Compatibility Mapping | กำหนดว่าสินค้าชิ้นไหนใช้ได้กับรถรุ่นไหน |
| AB-06 | Order Management | ดูรายการคำสั่งซื้อ กรองตามสถานะ อัปเดตสถานะการจัดส่ง |
| AB-07 | Coupon Management | สร้าง/แก้ไข/ลบคูปองส่วนลด (แบบเปอร์เซ็นต์หรือจำนวนเงิน) กำหนดวันหมดอายุและจำนวนใช้ได้ |
| AB-08 | Flash Sale Control | เปิด/ปิด Flash Sale กำหนดส่วนลดและเวลาสิ้นสุด |
| AB-09 | Dashboard & Reports | แดชบอร์ดแสดงยอดขาย/จำนวนออเดอร์/ลูกค้า กราฟยอดขายรายเดือน สินค้ายอดนิยม รายงานรายได้ตามช่องทางชำระเงิน |
| AB-10 | Store Settings | ตั้งค่าชื่อร้าน ข้อมูลติดต่อ สกุลเงิน |
| AB-11 | Maintenance Mode | เปิด/ปิดโหมดปิดปรับปรุง — เมื่อเปิด ลูกค้าจะเห็นหน้า "ปิดปรับปรุง" แต่แอดมินยังเข้าได้ |

## ความต้องการที่ไม่เกี่ยวข้องกับฟังก์ชัน (Non-Functional Requirements)

| หัวข้อ | ข้อกำหนด |
|--------|----------|
| ภาษา | รองรับไทย (ค่าเริ่มต้น) และอังกฤษ สลับได้ทันที |
| Performance | หน้าเว็บโหลดภายใน 3 วินาที, API ตอบกลับภายใน 500ms |
| Responsive | รองรับ Desktop, Tablet, Mobile |
| Data Integrity | ใช้ Prisma Transaction ป้องกันข้อมูลไม่สอดคล้อง (เช่น ตัดสต็อกพร้อมสร้างออเดอร์) |
| Design | Premium Dark Theme สไตล์ Performance/Racing |
